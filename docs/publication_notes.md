# Publication Notes

This repository is prepared for public release as a portfolio project and a
reproducible research prototype.

## Included

- source code for all pipeline phases
- unit tests
- public documentation
- source-connection notes
- synthetic example outputs under `examples/`

## Excluded

- real raw API responses
- processed parquet outputs
- backtest study folders
- generated paper drafts
- local SQLite databases
- model weights and caches
- local `.env` files

## Claims Policy

The project uses cautious terminology. It reports statistical candidates,
forecast failures, artifact risks, and known real-world shocks. It does not
claim proof of simulation, proof of rapid influx, or causal certainty.

The current research conclusion is:

> The system detects known real-world shocks and data artifacts, but does not
> currently identify an unexplained synchronized cross-domain continuity break.

## Optional LLM Components

Local LLM reports and paper drafting are optional presentation layers. They read
deterministic study outputs and should not be described as performing the
statistical analysis.
